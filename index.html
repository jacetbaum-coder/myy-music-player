The download server is a small program that runs on your computer and handles fetching music files.

To start it, open Terminal and run:

bash downloader/start.sh

Once it's running, click Test connection above.

Save to your library
Connect cloud storage so your music syncs across all your devices. Without this, downloads stay on this computer only.

1. Go to Cloudflare Dashboard → R2 Object Storage

2. Create a bucket (any name works, e.g. my-music)

3. Go to Manage R2 API Tokens → Create API token

4. Give it Object Read & Write access to your bucket

5. Copy the Account ID, Access Key ID, and Secret Access Key into the fields above

Your credentials stay in this browser only — they are never sent to our servers.
